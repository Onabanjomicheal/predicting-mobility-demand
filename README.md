## 📌 Overview

A machine learning project that predicts mobility demand and trip distribution patterns in urban areas. Applies regression and spatial analysis techniques to forecast where and when trips occur, supporting smarter transport planning.

**Tech Stack:** `Python` `Scikit-learn` `Pandas` `Urban Mobility` `Demand Forecasting`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/predicting-mobility-demand.git
cd predicting-mobility-demand
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
